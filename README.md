x# FitMate AI — Your Personalized Fitness & Nutrition Coach

FitMate AI is a modern, interactive fitness assistant that generates personalized calorie goals, macros, meal plans, workout routines, BMI insights, and logs workout sessions. It also includes a food image scanner and voice assistant using the Web Speech API.

## Features

1. User Profile System  
- Calculates BMR, TDEE, Target Calories, BMI with category.  
- Collects age, height, weight, sex, activity, and fitness goals.

2. Dynamic Meal Plan Generator  
- Auto-creates Breakfast, Lunch, Dinner, Snacks.  
- Based on your calorie target.

3. Personalized Workout Plan  
- Custom routines based on fitness goal and activity level.  
- Push/Pull/Legs, full-body, mixed training, rest day structure.

4. Advanced Workout Timer  
- Start / Pause / Reset  
- Logs each session  
- Critical countdown animation  
- Emoji-based timer states  
- Voice message when completed

5. Food Image Scanner (Simulated AI)  
- Upload food images  
- Detects common meals (apple, salad, burger, chicken, etc.)  
- Shows estimated calories, macros, suggestions  
- Includes voice feedback

6. Built-in Voice Assistant  
- Reads full fitness plan  
- Reads food scanner result  
- Uses browser speech synthesis

## Technologies Used

- HTML5 — layout, forms, structure  
- CSS3 — neon theme, glass UI, animations  
- JavaScript — logic, timers, planner, scanner  
- Web Speech API — voice assistant  
- File API — food scanner
